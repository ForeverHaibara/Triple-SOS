网页尝鲜版
更新:
1. 初步将 sos 程序迁移到本地网页版, 原先的单机版可能不再维护
优点: (1) 配方时不会卡住了 (2) 更易安装 (3) 更好开发 (4) 渲染LaTeX更快更好看且不用本地安装LaTeX
缺点: (1) 没怎么测试, 但是如果不是故意乱输应该问题不大 (2) 加载的时候需要有网络

2. 网页版的系数阵可以鼠标移动到某一项上查看次数。且等于零的地方改为浅灰色标识。

3. 网页版配方不会卡住还可以一次配多个, 缺点是你不知道出来的时候配的是哪个而且后面配的会覆盖掉前面的结果。

4. 现在输入会即时求出极小值点并输出推荐的切线（偶尔可能等两秒）。
采用真正的牛顿法计算又快又准, 原本的搜索步数设置已废弃。 如果需要修改输入的切线可以在框里直接修改。

5. 增加连分数逼近猜测小数, 猜测功力较强。
#####注意: 如果分数分子分母都极大(15位数/15位数那种), 那很可能是不严格相等的(约等于)!!#####

6. 加入了分式支持, 如果输入是分数则会自动通分并取分子 (不会看分母的正负的哦)



安装方式: 
分为两种情况 A, B:

A. 我已经安装过原先的配方器

    打开 cmd, 输入:
    pip install flask 
    
B. 我没有安装过原先的配方器
    安装 python-3.7 

    打开 cmd, 依次输入：
    pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/

    pip install sympy
    pip install numpy
    pip install scipy
    pip install matplotlib
    pip install flask


运行方式:
    每次运行都要做以下两步, 顺序不限:
    1. 右键文件夹里的 web_main.py 打开方式选择 python, 出现一个黑框不要关闭(可以最小化)
    2. 双击打开文件夹里的 triple.html (提示:可以在浏览器里收藏这个网页哦~)
